Το Tails OS περιλαμβάνει μια σειρά από εξειδικευμένες εφαρμογές για ασφαλή πλοήγηση, κρυπτογραφημένη επικοινωνία, διαχείριση αρχείων και ασφαλή διαγραφή δεδομένων. Όλα είναι προ-ρυθμισμένα με τις βέλτιστες προδιαγραφές ασφαλείας.

---

## 1. Tor Browser (Πλοήγηση στον Ιστό)

Ο **Tor Browser** είναι το κεντρικό εργαλείο του Tails. Βασίζεται στον Firefox ESR αλλά έχει τροποποιηθεί ριζικά για την προστασία της ανωνυμίας σας.

### Ρυθμίσεις Ασφαλείας (Security Levels):
Κάντε κλικ στο εικονίδιο με την **ασπίδα** στην πάνω δεξιά γωνία του Tor Browser και επιλέξτε *Settings*:
* **Standard (Προεπιλογή)**: Όλες οι λειτουργίες των ιστοσελίδων είναι ενεργές.
* **Safer**: Απενεργοποιεί τη Javascript σε μη-HTTPS ιστοσελίδες, απενεργοποιεί ορισμένες γραμματοσειρές και μαθηματικά σύμβολα.
* **Safest (Συνιστάται για μέγιστη ασφάλεια)**: Απενεργοποιεί πλήρως τη Javascript σε όλες τις ιστοσελίδες από προεπιλογή. Μπλοκάρει εικόνες SVG και ορισμένα media.

<Aside type="danger">
  **ΠΡΟΣΟΧΗ**: Μην εγκαθιστάτε πρόσθετα (Add-ons / Extensions) στον Tor Browser. Πρόσθετα όπως το AdBlock, μεταφραστές ή θέματα μπορούν να δημιουργήσουν ένα μοναδικό ψηφιακό αποτύπωμα (**Browser Fingerprint**) που θα σας ταυτοποιήσει ανάμεσα σε εκατομμύρια χρήστες Tor.
</Aside>

## 2. Kleopatra & GnuPG (Κρυπτογράφηση PGP)

Η **Kleopatra** είναι ο γραφικός διαχειριστής κλειδιών για το σύστημα **GnuPG (OpenPGP)**. Σας επιτρέπει να δημιουργείτε κλειδιά κρυπτογράφησης, να υπογράφετε κείμενα και να αποκρυπτογραφείτε αρχεία.

### Δημιουργία και Χρήση Κλειδιών:
<Steps>
1. Ανοίξτε το **Kleopatra** (*Applications > Office > Kleopatra*).
2. Κάντε κλικ στο **New Key Pair** (Νέο Ζεύγος Κλειδιών).
3. Επιλέξτε *Create a personal OpenPGP key pair*. Εισάγετε ένα ψευδώνυμο (δεν χρειάζεται το πραγματικό σας όνομα) και ένα email.
4. Ορίστε μια ισχυρή φράση πρόσβασης (passphrase) για να προστατεύσετε το ιδιωτικό σας κλειδί.
5. **Ανταλλαγή**: Μοιραστείτε το **Δημόσιο Κλειδί (Public Key)** σας με άλλους για να μπορούν να σας στέλνουν κρυπτογραφημένα μηνύματα. Κρατήστε το **Ιδιωτικό Κλειδί (Private Key)** αυστηρά μυστικό.
</Steps>

## 3. Metadata Cleaner (Καθαρισμός Μεταδεδομένων)

Κάθε αρχείο (φωτογραφία, PDF, έγγραφο Word) περιέχει κρυφά μεταδεδομένα (EXIF metadata) όπως η ημερομηνία δημιουργίας, το μοντέλο της κάμερας, το όνομα του υπολογιστή σας ή ακόμα και συντεταγμένες GPS.

* **Χρήση**: Ανοίξτε το **Metadata Cleaner** (*Applications > System Tools > Metadata Cleaner*).
* Σύρετε τα αρχεία σας μέσα στο παράθυρο.
* Το πρόγραμμα θα αναλύσει τα αρχεία και θα σας δείξει τι μεταδεδομένα περιέχουν.
* Κάντε κλικ στο **Clean** για να διαγράψετε οριστικά όλα τα μεταδεδομένα, παράγοντας ένα ασφαλές αρχείο για διαμοιρασμό.

## 4. OnionShare (Ασφαλής Διαμοιρασμός)

Το **OnionShare** σας επιτρέπει να μοιράζεστε αρχεία, να φιλοξενείτε ιστοσελίδες ή να δημιουργείτε chat rooms χρησιμοποιώντας το δίκτυο Tor ως προσωρινό server, χωρίς να αποκαλύπτετε την IP σας.

* **Διαμοιρασμός Αρχείων (Share Files)**: Επιλέξτε τα αρχεία που θέλετε να στείλετε. Το OnionShare θα δημιουργήσει μια τυχαία διεύθυνση `.onion` (π.χ. `http://onionshare-xyz.onion`). Στείλτε αυτή τη διεύθυνση στον παραλήπτη. Μπορεί να κατεβάσει τα αρχεία ανοίγοντας τη διεύθυνση στον δικό του Tor Browser. Μόλις κλείσετε το OnionShare, η σελίδα παύει να υπάρχει.
* **Anonymous Chat**: Δημιουργεί ένα ασφαλές δωμάτιο συνομιλίας προσβάσιμο μόνο μέσω Tor Browser, χωρίς να αποθηκεύεται κανένα ιστορικό σε κανέναν server.

## 5. Secrets (Διαχείριση Κωδικών KeePassXC)

Το **Secrets** είναι ένας offline διαχειριστής κωδικών πρόσβασης συμβατός με το πρότυπο **KeePassXC**.

* **Αποθήκευση**: Δημιουργεί μια κρυπτογραφημένη βάση δεδομένων (`.kdbx`) όπου μπορείτε να αποθηκεύετε όλους τους κωδικούς σας.
* **Ασφάλεια**: Μπορείτε να αποθηκεύσετε τη βάση δεδομένων στον φάκελο `Persistent` για να την έχετε διαθέσιμη σε κάθε boot.
* **Password Generator**: Περιλαμβάνει γεννήτρια ισχυρών, τυχαίων κωδικών πρόσβασης.

## 6. Secure Deletion (Ασφαλής Διαγραφή Αρχείων)

Όταν διαγράφετε ένα αρχείο σε ένα κανονικό λειτουργικό σύστημα (ακόμα και από τον κάδο ανακύκλωσης), τα δεδομένα παραμένουν στον δίσκο και απλά ο χώρος σημαίνεται ως "ελεύθερος για εγγραφή". 

Στο Tails, εάν θέλετε να διαγράψετε οριστικά ένα αρχείο από το Persistent Storage ή από ένα εξωτερικό USB drive:
1. Ανοίξτε το πρόγραμμα διαχείρισης αρχείων (Files).
2. Κάντε δεξί κλικ στο αρχείο που θέλετε να διαγράψετε.
3. Επιλέξτε **Wipe** (Ασφαλής Διαγραφή).
4. Το Tails θα γράψει τυχαία δεδομένα (zeroes/random bits) πάνω στη φυσική τοποθεσία του αρχείου στον δίσκο πολλές φορές, καθιστώντας την ανάκτησή του αδύνατη.

## 7. Electrum (Bitcoin Wallet)

Το **Electrum** είναι ένα ελαφρύ και ασφαλές πορτοφόλι Bitcoin.
* **Προ-ρυθμισμένο**: Συνδέεται αυτόματα στο δίκτυο Bitcoin μέσω του δικτύου Tor για να προστατεύσει την IP σας.
* **Persistent**: Μπορείτε να ενεργοποιήσετε το Electrum προφίλ στο Persistent Storage ώστε να αποθηκεύεται το πορτοφόλι σας.
* **Συμβουλή Ασφαλείας**: Κατά τη δημιουργία του πορτοφολιού, γράψτε τη seed phrase (φράση ανάκτησης 12 λέξεων) σε ένα χαρτί και φυλάξτε τη σε ασφαλές μέρος. Ποτέ μην την αποθηκεύετε σε ψηφιακή μορφή χωρίς ισχυρή κρυπτογράφηση.
