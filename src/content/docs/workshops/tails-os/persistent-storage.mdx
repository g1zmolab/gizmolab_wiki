Από προεπιλογή, το Tails OS είναι "αμνησιακό" και διαγράφει τα πάντα κατά τον τερματισμό. Εάν θέλετε να αποθηκεύετε μόνιμα αρχεία, ρυθμίσεις ή εφαρμογές, πρέπει να ενεργοποιήσετε το **Persistent Storage** (Μόνιμος Χώρος Αποθήκευσης).

---

## 1. Τεχνολογία και Ασφάλεια

Ο Μόνιμος Χώρος Αποθήκευσης δημιουργείται ως ένα ξεχωριστό διαμέρισμα (partition) στο USB stick σας.
* **Κρυπτογράφηση LUKS**: Το διαμέρισμα κρυπτογραφείται με **LUKS (Linux Unified Key Setup)**, το οποίο αποτελεί το πρότυπο κρυπτογράφησης δίσκων στο Linux.
* **Ασφάλεια**: Η πρόσβαση στα αρχεία είναι αδύνατη χωρίς τη σωστή φράση πρόσβασης (passphrase). Ακόμα κι αν χάσετε το USB stick, κανείς δεν μπορεί να διαβάσει τα δεδομένα σας χωρίς τον κωδικό.
* **Απομόνωση**: Η κρυπτογράφηση ξεκλειδώνει χειροκίνητα κατά την εκκίνηση στην Οθόνη Καλωσορίσματος. Αν δεν την ξεκλειδώσετε, το Tails συμπεριφέρεται ως κανονικό αμνησιακό σύστημα και το Persistent Storage παραμένει κλειδωμένο και αόρατο.

---

## 2. Δημιουργία του Persistent Storage

<Steps>
1. **Άνοιγμα της Εφαρμογής**:
   Στο desktop του Tails, επιλέξτε *Applications > System Tools > Persistent Storage*.

2. **Ορισμός Φράσης Πρόσβασης (Passphrase)**:
   Επιλέξτε μια ισχυρή φράση πρόσβασης. Συνιστάται μια πρόταση με πολλές τυχαίες λέξεις (passphrase) παρά ένας απλός κωδικός.
   <Aside type="danger">
     **ΠΡΟΣΟΧΗ**: Αν ξεχάσετε τη φράση πρόσβασης, είναι αδύνατο να ανακτήσετε τα αρχεία σας. Δεν υπάρχει επιλογή "ξέχασα τον κωδικό μου" ή ανάκτησης!
   </Aside>

3. **Επιλογή Στοιχείων προς Αποθήκευση**:
   Το Tails σας επιτρέπει να επιλέξετε ακριβώς τι θα αποθηκεύεται μόνιμα. Ενεργοποιήστε μόνο όσα χρειάζεστε:
   * **Personal Files (Προσωπικά Αρχεία)**: Δημιουργεί έναν φάκελο `Persistent` στο home directory σας. Οτιδήποτε αποθηκεύετε εκεί παραμένει μόνιμα.
   * **Welcome Screen**: Αποθηκεύει τις προτιμήσεις σας για τη γλώσσα και το MAC spoofing.
   * **Network Connections**: Αποθηκεύει κωδικούς Wi-Fi.
   * **GnuPG (Kleopatra)**: Αποθηκεύει τα OpenPGP κλειδιά σας.
   * **Thunderbird**: Αποθηκεύει τα email και τις ρυθμίσεις σας.
   * **Tor Browser Bookmarks**: Αποθηκεύει τους σελιδοδείκτες σας.
   * **SSH Client & Keys**: Αποθηκεύει τα SSH κλειδιά και τις ρυθμίσεις σύνδεσης.
   * **Additional Software**: Αποθηκεύει πρόσθετα πακέτα λογισμικού (δείτε παρακάτω).

4. **Αποθήκευση**:
   Κάντε κλικ στο κουμπί **Create** και περιμένετε να ολοκληρωθεί η διαδικασία.
</Steps>

## 3. Ξεκλείδωμα και Χρήση

Την επόμενη φορά που θα εκκινήσετε το Tails, στην Οθόνη Καλωσορίσματος (Welcome Screen) θα εμφανιστεί ένα πεδίο εισαγωγής κωδικού:
1. Εισάγετε τη φράση πρόσβασης στο πεδίο **Persistent Storage**.
2. Κάντε κλικ στο **Unlock**.
3. Πατήστε **Start Tails** για να εισέλθετε στο σύστημα με ξεκλειδωμένο τον προσωπικό σας χώρο.
4. Θα βρείτε τον φάκελο των προσωπικών σας αρχείων πηγαίνοντας στο *Files > Persistent*.

---

## 4. Εγκατάσταση Πρόσθετου Λογισμικού (Additional Software)

Το Tails περιλαμβάνει μια επιλογή βασικών εφαρμογών, αλλά μπορείτε να εγκαταστήσετε επιπλέον λογισμικό από τα επίσημα αποθετήρια του Debian (μέσω του Synaptic Package Manager ή του τερματικού).

### Πώς να κάνετε τις εφαρμογές μόνιμες:
<Steps>
1. Βεβαιωθείτε ότι έχετε ξεκλειδώσει το Persistent Storage και έχετε ενεργοποιήσει την επιλογή **Additional Software** στις ρυθμίσεις του.
2. Εγκαταστήστε την εφαρμογή χρησιμοποιώντας το τερματικό. Για παράδειγμα, για να εγκαταστήσετε τον text editor `Vim`:
   ```bash
   sudo apt update
   sudo apt install vim
   ```
   *(Απαιτείται να έχετε ορίσει Administration Password στην Welcome Screen)*.
3. Μόλις ολοκληρωθεί η εγκατάσταση, το Tails θα εμφανίσει μια ειδοποίηση ρωτώντας αν θέλετε να εγκαθίσταται αυτόματα αυτή η εφαρμογή σε κάθε εκκίνηση. Πατήστε **Install Every Time**.
4. Σε κάθε επόμενη εκκίνηση (εφόσον ξεκλειδώνετε το Persistent Storage), το Tails θα κατεβάζει και θα εγκαθιστά αυτόματα την εφαρμογή στο υπόβαθρο, χωρίς να χρειάζεται να την ξαναγράψετε.
</Steps>

## 5. Αντίγραφα Ασφαλείας (Backups)

<Aside type="danger">
  Τα USB sticks έχουν περιορισμένο κύκλο ζωής και μπορούν να καταστραφούν ή να χαθούν εύκολα. Είναι **απαραίτητο** να κρατάτε αντίγραφα ασφαλείας του Persistent Storage!
</Aside>

Ο ασφαλέστερος τρόπος για backup είναι να δημιουργήσετε ένα δεύτερο Tails USB stick και να αντιγράψετε τα δεδομένα σας:

<Steps>
1. Δημιουργήστε ένα δεύτερο bootable Tails USB stick (σύμφωνα με τον Οδηγό Εγκατάστασης).
2. Εκκινήστε το Tails από το **πρώτο (κύριο)** USB stick σας και ξεκλειδώστε το Persistent Storage.
3. Συνδέστε το **δεύτερο (εφεδρικό)** USB stick στον υπολογιστή.
4. Ανοίξτε την εφαρμογή **Tails Cloner** (*Applications > System Tools > Tails Cloner*).
5. Επιλέξτε **Clone current Tails and copy Persistent Storage** (Κλωνοποίηση του τρέχοντος Tails και αντιγραφή του Μόνιμου Χώρου).
6. Επιλέξτε το εφεδρικό USB stick ως προορισμό και ξεκινήστε τη διαδικασία. Όλο το Persistent Storage θα αντιγραφεί κρυπτογραφημένο στο νέο stick.
</Steps>
