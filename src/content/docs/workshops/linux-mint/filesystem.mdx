## Εισαγωγή

Στο Linux Mint (και γενικά στο Linux) το σύστημα αρχείων είναι διαφορετικό από αυτό των Windows. Δεν υπάρχουν "C:" ή "D:" δίσκοι — όλα ξεκινούν από τη ρίζα, που συμβολίζεται με το `/`. Κάθε αρχείο και φάκελος στο σύστημα βρίσκεται κάπου κάτω από αυτό το κεντρικό **root directory**.

---

## Βασική Δομή

```
/
├── bin        → Βασικά εκτελέσιμα (π.χ. cp, mv, ls)
├── boot       → Αρχεία εκκίνησης του συστήματος
├── dev        → Συσκευές (π.χ. δίσκοι, USB, κάμερες)
├── etc        → Ρυθμίσεις συστήματος και εφαρμογών
├── home       → Προσωπικοί φάκελοι χρηστών
│   ├── user1
│   └── user2
├── lib        → Κοινόχρηστες βιβλιοθήκες
├── media      → Αυτόματη προσάρτηση εξωτερικών δίσκων
├── mnt        → Προσωρινή προσάρτηση δίσκων
├── opt        → Επιπλέον προγράμματα
├── root       → Ο προσωπικός φάκελος του διαχειριστή
├── sbin       → Εκτελέσιμα για τον διαχειριστή
├── tmp        → Προσωρινά αρχεία
├── usr        → Εφαρμογές και βιβλιοθήκες χρήστη
└── var        → Μεταβαλλόμενα δεδομένα (logs, cache, mail)
```

---

## Ο Φάκελος `/home`

Είναι ο πιο σημαντικός για τον καθημερινό χρήστη. Εκεί αποθηκεύονται τα έγγραφα, οι ρυθμίσεις και όλα τα προσωπικά δεδομένα κάθε χρήστη.

<Aside type="tip">
Αν κάνεις επανεγκατάσταση του Linux, μπορείς να κρατήσεις τον φάκελο `/home` (αν βρίσκεται σε ξεχωριστό partition) ώστε να μη χαθούν οι ρυθμίσεις σου.
</Aside>

---

## Ο Φάκελος `/etc`

Περιέχει τα αρχεία ρυθμίσεων για όλο το σύστημα — κάτι σαν τον "Πίνακα Ελέγχου" αλλά σε μορφή αρχείων κειμένου. Για παράδειγμα, το αρχείο `/etc/hosts` καθορίζει τοπικά ονόματα δικτύου.

---

## Ο Φάκελος `/usr`

Εδώ αποθηκεύονται τα περισσότερα προγράμματα και βιβλιοθήκες που χρησιμοποιούν οι χρήστες. Αν εγκαταστήσεις ένα πρόγραμμα μέσω `apt` ή `flatpak`, πιθανότατα θα το βρεις μέσα στο `/usr`.

---

## Εντολές Εξερεύνησης

| Εντολή | Λειτουργία |
|---|---|
| `pwd` | Εμφανίζει τον τρέχοντα φάκελο |
| `ls` | Λίστα περιεχομένων |
| `cd /home` | Μετακίνηση σε φάκελο |
| `cd ..` | Επιστροφή πίσω |
| `tree` | Εμφάνιση δομής (αν είναι εγκατεστημένο) |

<Aside type="caution">
Το Linux διαχωρίζει τα αρχεία **συστήματος** από τα αρχεία **χρήστη**. Μην αλλάζεις τίποτα εκτός του `/home` εκτός αν ξέρεις τι κάνεις.
</Aside>
