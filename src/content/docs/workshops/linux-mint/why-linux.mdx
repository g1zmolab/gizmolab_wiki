## Γιατί να φύγεις από τα Windows

### Τέλος Υποστήριξης Windows 10

Στις **14 Οκτωβρίου 2025** τερματίστηκε η επίσημη υποστήριξη των Windows 10 από τη Microsoft. Αυτό σημαίνει: κανένα patch ασφαλείας, κανένα update, κανένα bugfix — παρά μόνο μέσω του πληρωμένου προγράμματος Extended Security Updates (ESU), που είναι προσωρινό και όχι δωρεάν. Ένα μηχάνημα με Windows 10 χωρίς ESU είναι:

- Ευάλωτο σε κυβερνοεπιθέσεις που εκμεταλλεύονται νέα, μη διορθωμένα κενά ασφαλείας
- Εκτεθειμένο σε malware και ιούς χωρίς προστασία
- Σε κίνδυνο για τα προσωπικά δεδομένα του χρήστη

### Μαζική Παρακολούθηση

Τα σύγχρονα Windows συλλέγουν συστηματικά δεδομένα χρήσης:

| Κατηγορία | Τι συλλέγεται |
|---|---|
| Ιστορικό Χρήσης | Κάθε κλικ, κάθε αναζήτηση, κάθε site που επισκέπτεσαι |
| Γεωγραφική Θέση | Πού βρίσκεσαι, πού πας, πού έχεις πάει |
| Προσωπικά Αρχεία | Φωτογραφίες, έγγραφα, επικοινωνίες |
| Συμπεριφορά | Πληκτρολογήσεις, φωνητικές εντολές, συνήθειες |

### Εμπορευματοποίηση των Δεδομένων σου

- **Στοχευμένες Διαφημίσεις**: Τα δεδομένα σου τροφοδοτούν διαφημίσεις μέσα στο ίδιο το λειτουργικό σύστημα.
- **Πώληση σε Τρίτους**: Η Microsoft κερδοφορεί πουλώντας προσωπικά δεδομένα χρηστών σε τρίτους.
- **Παροχή σε Κρατικές Αρχές**: Χωρίς πάντα ρητή συναίνεση, δεδομένα μοιράζονται με αρχές.

### Υποχρεωτική Αναβάθμιση Hardware

Τα Windows 11 απαιτούν νέο, συχνά ακριβό εξοπλισμό:

- TPM 2.0 chip
- UEFI Secure Boot
- Συγκεκριμένους, σχετικά πρόσφατους επεξεργαστές
- Περισσότερη RAM και αποθηκευτικό χώρο

Αποτέλεσμα: απόλυτα λειτουργικοί υπολογιστές καταλήγουν στα σκουπίδια ως **ηλεκτρονικά απόβλητα**, απλά και μόνο επειδή δεν πληρούν τεχνητά κριτήρια hardware. Το Linux Mint, αντίθετα, τρέχει άνετα σε μηχανήματα 10+ ετών.

## Εισαγωγή στο Linux

### Τι είναι το Linux;

Το Linux είναι ένα **ελεύθερο λειτουργικό σύστημα ανοιχτού κώδικα** που δημιουργήθηκε και αναπτύσσεται συλλογικά από χιλιάδες προγραμματιστές σε όλο τον κόσμο. Δεν ανήκει σε καμία εταιρεία — ανήκει στην κοινότητα.

### Φιλοσοφία Ελευθερίας

| Αρχή | Περιγραφή |
|---|---|
| Ελεύθερο | Κανένα κόστος, καμία συνδρομή, καμία άδεια |
| Ανοιχτό | Ο κώδικας διαθέσιμος σε όλους για έλεγχο και βελτίωση |
| Διαφανές | Χωρίς κρυφές λειτουργίες παρακολούθησης |
| Συλλογικό | Από την κοινότητα, για την κοινότητα |

### Πλεονεκτήματα

- Απόλυτη προστασία της ιδιωτικότητάς σου
- Ασφάλεια χωρίς ιούς και malware με τη λογική των Windows (η αρχιτεκτονική δικαιωμάτων του Linux καθιστά πολύ πιο δύσκολη τη μαζική εξάπλωση malware)
- Ταχύτητα ακόμα και σε παλαιότερα μηχανήματα
- Σταθερότητα χωρίς crashes και αναγκαστικές επανεκκινήσεις για updates
- Εκατοντάδες δωρεάν προγράμματα, εύκολα προσβάσιμα

### Linux στην Καθημερινότητα

Το Linux είναι παντού, ακόμα κι αν δεν το βλέπεις:

| Πού | Πώς |
|---|---|
| Smartphones | Το Android βασίζεται στον πυρήνα Linux |
| Servers | Το μεγαλύτερο μέρος του διαδικτύου τρέχει σε Linux servers |
| Supercomputers | Το σύνολο σχεδόν των 500 ισχυρότερων υπερυπολογιστών τρέχει 100% Linux |

## Γιατί Linux Mint

### Η Ιδανική Διανομή για Αρχάριους

Το Linux Mint έχει σχεδιαστεί ειδικά για όσους έρχονται από Windows: **οικείο περιβάλλον εργασίας**, με φιλική διεπαφή που θυμίζει τη λογική των Windows — δεν χρειάζεται να ξεμάθεις τίποτα από το μηδέν.

| Αρχή | Περιγραφή |
|---|---|
| Εύκολο | Φιλική διεπαφή, μηδενική καμπύλη εκμάθησης |
| Πλήρες | Έρχεται με όλα τα απαραίτητα προγράμματα προεγκατεστημένα |
| Σταθερό | Βασισμένο στο Ubuntu (και μέσω αυτού στο Debian), εξασφαλίζει αξιοπιστία |
| Σεβασμός | Καμία τηλεμετρία, καμία παρακολούθηση |

### Τι Περιλαμβάνει

| Component | Χρήση |
|---|---|
| Cinnamon Desktop | Κομψό και γνώριμο περιβάλλον εργασίας |
| LibreOffice | Πλήρης σουίτα γραφείου (αντίστοιχο Word, Excel, PowerPoint) |
| Firefox | Ασφαλής περιήγηση στο διαδίκτυο |
| Thunderbird | Διαχείριση email |
| Multimedia codecs | Αναπαραγωγή μουσικής και βίντεο out-of-the-box |
| Software Manager | Χιλιάδες δωρεάν εφαρμογές με ένα κλικ |

<Aside type="tip">
Πέρα από την έκδοση **Cinnamon** (η πιο δημοφιλής, αυτή που καλύπτει ο οδηγός), το Linux Mint διατίθεται και σε εκδόσεις **MATE** και **Xfce** — πιο ελαφριές, καλή επιλογή για πολύ παλιά μηχανήματα με περιορισμένη RAM.
</Aside>
