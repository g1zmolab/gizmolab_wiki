## Εκκίνηση από το USB

<Steps>

1. Επανεκκίνησε τον υπολογιστή με το USB συνδεδεμένο.

2. Μπες στο boot menu (`F12`/`Esc`/κ.λπ. — βλ. ενότητα "Προετοιμασία") και επίλεξε το USB stick.

3. Θα εμφανιστεί το μενού του GRUB bootloader — επίλεξε **"Start Linux Mint"**.

</Steps>

---

## Live Session (δοκιμαστική λειτουργία)

Το Linux Mint ξεκινά σε **live mode** — τρέχει εξ ολοκλήρου από το USB, χωρίς να αγγίξει τον δίσκο σου, ώστε να μπορείς να το δοκιμάσεις πρώτα (Wi-Fi, γραφικά, ήχος) πριν αποφασίσεις να το εγκαταστήσεις μόνιμα.

<Aside type="tip">
Αξιοποίησε τη live session για να ελέγξεις ότι όλο το hardware λειτουργεί σωστά πριν προχωρήσεις στην εγκατάσταση.
</Aside>

## Βήματα του installer

Στην επιφάνεια εργασίας, διπλό κλικ στο εικονίδιο **"Install Linux Mint"**.

<Steps>

1. **Γλώσσα**: Επίλεξε Ελληνικά (ή όποια προτιμάς).

2. **Πληκτρολόγιο**: Επιβεβαίωσε τη διάταξη (π.χ. Greek).

3. **Multimedia codecs**: Επίλεξε "Install multimedia codecs" ώστε βίντεο/ήχος να δουλεύουν σωστά εξαρχής (χρειάζεται σύνδεση internet).

4. **Τύπος εγκατάστασης** — το πιο κρίσιμο βήμα:

   | Επιλογή | Πότε τη διαλέγεις |
   |---|---|
   | Install alongside Windows | Θέλεις dual-boot — να κρατήσεις και τα Windows. Ο installer δημιουργεί αυτόματα νέο partition |
   | Erase disk and install | Θέλεις να αντικαταστήσεις πλήρως το υπάρχον λειτουργικό |
   | Something else | Προχωρημένοι χρήστες — πλήρης έλεγχος πάνω στα partitions |

   <Aside type="caution">
   Η επιλογή "Erase disk" διαγράφει τα πάντα στον δίσκο. Βεβαιώσου ότι έχεις backup.
   </Aside>

5. Αν διάλεξες dual-boot: χρησιμοποίησε το slider για να ορίσεις πόσο χώρο θα πάρει το Linux Mint (προτεινόμενο ελάχιστο: 30-40GB, ιδανικά περισσότερο).

6. **Ζώνη ώρας**: Athens/Europe.

7. **Λογαριασμός χρήστη**: Όνομα, username (χωρίς κεφαλαία/τόνους/κενά), κωδικός. Επίλεξε αν θέλεις αυτόματη σύνδεση ή σύνδεση με κωδικό.

   <Aside type="tip">
   Η αυτόματη σύνδεση δεν προτείνεται αν ο υπολογιστής δεν είναι μόνο δικός σου.
   </Aside>

8. Ο installer αντιγράφει τα αρχεία (10-20 λεπτά ανάλογα με το μηχάνημα).

9. Στο τέλος, θα σου ζητηθεί επανεκκίνηση — αφαίρεσε το USB stick όταν σου το ζητήσει η οθόνη.

</Steps>

---

## Πρώτη εκκίνηση

Αν επέλεξες dual-boot, θα δεις τώρα το μενού **GRUB** σε κάθε εκκίνηση, όπου διαλέγεις ανάμεσα σε Linux Mint και Windows.
