## Πριν την εγκατάσταση

### Ελάχιστες απαιτήσεις συστήματος (Linux Mint 22.3)

| Πόρος | Ελάχιστο | Προτεινόμενο |
|---|---|---|
| RAM | 2 GB | 4 GB+ |
| Αποθηκευτικός χώρος | 20 GB | 100 GB+ |
| Επεξεργαστής | Οποιοσδήποτε x86_64 (64-bit) των τελευταίων ~15 ετών | — |
| Ανάλυση οθόνης | 1024x768 | — |

<Aside type="tip">
Πρακτικά: αν ένα μηχάνημα έτρεχε αξιοπρεπώς Windows 7/8/10, θα τρέξει το Linux Mint άνετα — συχνά πολύ πιο γρήγορα απ' ό,τι έτρεχε τα Windows.
</Aside>

### Backup πρώτα — πάντα

<Aside type="caution">
Πριν αγγίξεις partitions ή κάνεις οτιδήποτε στον δίσκο, βεβαιώσου ότι έχεις backup.
</Aside>

- Αντίγραφο ασφαλείας όλων των προσωπικών αρχείων (Έγγραφα, Φωτογραφίες, Λήψεις) σε εξωτερικό δίσκο ή cloud
- Σημείωσε τυχόν κωδικούς/license keys προγραμμάτων που χρησιμοποιείς
- Αν θα κρατήσεις dual-boot με Windows: βεβαιώσου ότι τα Windows είναι πλήρως ενημερωμένα και δεν βρίσκονται σε κατάσταση "Fast Startup" (Πίνακας Ελέγχου → Επιλογές τροφοδοσίας → Επιλογή λειτουργίας κουμπιών τροφοδοσίας → απενεργοποίησε "Γρήγορη εκκίνηση") — η γρήγορη εκκίνηση των Windows μπορεί να κλειδώσει το NTFS partition και να προκαλέσει προβλήματα πρόσβασης από το Linux.

### BIOS/UEFI ρυθμίσεις

Τα περισσότερα σύγχρονα μηχανήματα χρησιμοποιούν **UEFI** αντί για το παλιό BIOS. Πριν την εγκατάσταση:

<Steps>

1. Μπες στις ρυθμίσεις UEFI/BIOS (συνήθως πατώντας `F2`, `F10`, `F12`, `Del` ή `Esc` κατά την εκκίνηση — διαφέρει ανά κατασκευαστή).

2. **Secure Boot**: Το σύγχρονο Linux Mint installer υποστηρίζει Secure Boot out-of-the-box (μέσω signed shim bootloader) — δεν χρειάζεται συνήθως να το απενεργοποιήσεις. Αν αντιμετωπίσεις προβλήματα εκκίνησης από το USB, δοκίμασε να το απενεργοποιήσεις προσωρινά ως fallback.

3. **Boot Order**: Βεβαιώσου ότι το USB stick έχει προτεραιότητα εκκίνησης, ή χρησιμοποίησε το one-time boot menu (συνήθως `F12` ή `Esc`) για να επιλέξεις απευθείας το USB χωρίς μόνιμη αλλαγή στη σειρά εκκίνησης.

</Steps>

## Δημιουργία USB εγκατάστασης

Χρειάζεσαι ένα USB stick **τουλάχιστον 4GB** (θα διαγραφούν όλα τα δεδομένα του).

### Μέθοδος 1: Rufus (Windows)

<Steps>

1. Κατέβασε το Linux Mint ISO και το Rufus από τα επίσημα links παραπάνω.

2. Σύνδεσε το USB stick.

3. Άνοιξε το Rufus — θα ανιχνεύσει αυτόματα το USB.

4. Στο **"Boot selection"**, πάτα `SELECT` και διάλεξε το αρχείο `.iso` που κατέβασες.

5. Άφησε τις υπόλοιπες ρυθμίσεις στις προεπιλογές (Partition scheme: `GPT` για σύγχρονα UEFI συστήματα, File system: `FAT32`).

6. Πάτα `START`. Το Rufus μπορεί να ρωτήσει αν θέλεις "ISO Image mode" ή "DD Image mode" — επίλεξε **ISO Image mode**.

7. Περίμενε να ολοκληρωθεί η διαδικασία (συνήθως 5-15 λεπτά ανάλογα με το USB).

</Steps>

### Μέθοδος 2: Ventoy (εναλλακτικό, πολλαπλά ISO)

<Steps>

1. Κατέβασε και εγκατέστησε το Ventoy στο USB (μία φορά, μέσω `Ventoy2Disk`).

2. Μετά την πρώτη εγκατάσταση, απλά **αντιγράφεις** το αρχείο `.iso` του Linux Mint (και όποιο άλλο ISO θες) απευθείας μέσα στο USB, σαν κανονικό αρχείο — δεν χρειάζεται να ξαναγράψεις το USB κάθε φορά που αλλάζεις ISO.

3. Κατά την εκκίνηση από το USB, το Ventoy θα σου δείξει μενού επιλογής ανάμεσα στα ISO που έχεις αποθηκεύσει.

</Steps>
